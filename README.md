# WellStat-Health-Checker
A **Python object-oriented application** for managing health data, emergencies, and basic illness tracking.   This app demonstrates good OOP principles while providing a functional GUI for users to interact with health records and emergency contacts. This OOP Project has not being deployed yet,but can perform the following tasks:


- Track personal health information and symptoms
- Manage patient records
- Set up emergency contacts and send alerts
- Predict possible illnesses based on symptoms
- Learn object-oriented programming in Python through a practical example

The app uses **Tkinter** for GUI and a modular, class-based backend for easy maintenance and future expansion.



## **Features**

- **User Management**: Add and view users/patients.
- **Symptom Tracking**: Record symptoms for patients.
- **Illness Prediction**: Match patient symptoms to common illnesses.
- **Emergency Alerts**: Notify emergency contacts with messages.
- **GUI Interface**: User-friendly Tkinter-based interface.
